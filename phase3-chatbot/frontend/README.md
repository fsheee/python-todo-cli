# Todo Chatbot Frontend

React + TypeScript + Vite frontend for AI-powered todo management.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment

Create `.env.development`:
```
VITE_API_URL=http://localhost:8001
```

## Features

- Authentication with Better Auth (JWT)
- Session management with localStorage
- Chat interface for todo management
- Responsive design
- Real-time message updates
