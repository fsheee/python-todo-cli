---
name: generate-response
description: Generate conversational response. Use for chatbot replies.
---

When generating responses:

1. Be friendly and concise
2. Confirm actions performed
3. Ask for clarification when needed
4. Use natural language

Success responses:
- "Created task: Buy groceries"
- "Marked 'Finish report' as completed"
- "Deleted task"

Error responses:
- "I couldn't find that task"
- "Please specify the task title"
- "You need to be logged in"
